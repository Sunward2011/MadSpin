# MadSpin

A Minecraft mod for Forge 1.20.1 that makes your character spin in third-person view while the camera stays still.

## Features

- **Spin Mode**: Player model rotates continuously while you control the camera
- **Back Mode**: Player model faces backward with configurable head/arm angles
- **Head Angle Control**: Adjust head pitch independently per mode
- **Arm Angle Control**: Arms raise/lower when holding items
- **Third-person Crosshair**: Crosshair appears in third-person view
- **Persistent Config**: All settings saved to `config/madspin-client.toml`

## Controls

| Key | Action |
|-----|--------|
| `M` | Open configuration GUI |
| `X` (default, customizable) | Toggle spin on/off |

## Requirements

- Minecraft 1.20.1
- Forge 47.3.0 or later
- Java 17

## Installation

1. Download the latest `.jar` from [Releases](https://github.com/Sunward2011/MadSpin/releases)
2. Place it in your `mods` folder
3. Launch Minecraft with Forge
